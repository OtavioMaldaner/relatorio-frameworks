Laravel

1- Comece criando um novo projeto no terminal, execute o seguinte comando para criar um novo projeto Laravel chamado "TaskManager":

composer create-project --prefer-dist laravel/laravel TaskManager

2- Execute o comando a seguir para uma migração que criará a tabela de tarefas

php artisan make:migration create_tasks_table --create=tasks

3- Abra o arquivo gerado em ‘database/migrations/YYYY_MM_DD_HHmmss_create_task_table.php’ e adicione o seguinte código:

public function up()
            {
                Schema::create('tasks', function (Blueprint $table) {
                    $table->id();
                    $table->string('title');
                    $table->timestamps();
                });
            }

4- Em seguida, execute a migração para criar a tabela no banco de dados: ‘php artisan migrate’

5- Execute o seguinte comando no terminal para gerar um modelo para a tabela de tarefas:

‘php artisan make:model Task’

6- Abra o arquivo routes/web.php e adicione a seguinte rota:

use App\Http\Controllers\TaskController;

Route::get('/tasks', [TaskController::class, 'index']);

7- Em seguida, execute o seguinte comando no terminal para gerar um controlador para lidar com as tarefas:

‘php artisan make:controller TaskController --resource’

8- Abra o arquivo app/Http/Controllers/TaskController.php e atualize o código do controlador da seguinte forma:

namespace App\Http\Controllers;

use App\Models\Task; 

class TaskController extends Controller
            {
                public function index()
                {
                    $tasks = Task::all();
                    return view('tasks.index', compact('tasks'));
                }
            }

9- Crie um diretório chamado tasks dentro do diretório resources/views e crie um arquivo de view chamado index.blade.php.

No arquivo index.blade.php, adicione o seguinte código:

<!DOCTYPE html>
        <html>
        <head>
            <title>Lista de Tarefas</title>
        </head>
        <body>
            <h1>Lista de Tarefas</h1>
        
            <ul>
                @foreach ($tasks as $task)
                    <li>{{ $task->title }}</li>
                @endforeach
            </ul>
        </body>
        </html>

10- No terminal, navegue até a pasta do projeto (cd TaskManager) e execute o seguinte comando para iniciar o servidor de desenvolvimento do Laravel:

php artisan serve

11- Abra seu navegador e acesse http://localhost:8000/tasks. Você verá uma lista de tarefas (atualmente vazia) na página.
