Para este exemplo, foi criado um projeto utilizando ViteJS

Para criar o projeto, você deve executar o comando npm create vite@latest para criar um projeto com a versão mais recente da ferramenta. (Necessita de NodeJS 14.8+)

Em seguida adicione o nome que você deseja ao seu projeto, o framework React, e, em seguida, a variante que desejar. Neste caso, será utilizada a variante com TypeScript.

Com isso pronto, basta entrar na pasta que foi criada o projeto e executar o comando npm install e sem seguida npm run dev, que o código estará executando em um servidor local.

Caso você vá ao arquivo App.tsx, que retorna uma função com um código HTML, você deve encontrar um código parecido com este:

Este código inicia importando o hook useState, que armazena informações temporariamente, as logos do React e do Vite e o código css.

Em seguida, é criada a função que retornará o que será exibido na tela. Já na primeira linha temos o "state" que controla um contador básico que a tela possui. o state "count" possui um valor inicial 0.

Após isso, dentro do return(), temos a criação da página, com as logos que foram importadas no início, juntamente com o nome de cada organização.

Em baixo da apresentação das organização, está disposto um card, que possui o contador que é controlado pelo hook importado inicialmente. Assim, como definido na função, em cada clique será somado 1 ao valor antigo do state.

Por fim, a aplicação instrui a editar o código e clicar nas logos para aprender a dominar melhor a ferramenta.
