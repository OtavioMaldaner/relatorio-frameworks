Relatório Frameworks

1. O que são Frameworks e para que são utilizados?

Um framework é uma estrutura fundamental para desenvolver aplicações web com propósitos específicos, oferecendo uma base sólida que pode reduzir o custo e os problemas associados ao desenvolvimento.

Vamos pensar que um frame representa a estrutura de uma casa, na qual só existem as paredes de tijolos levantadas ou mesmo um carro somente com a lataria, como vemos nas montadoras. Essas estruturas estão aguardando o desenvolvimento (work), que no exemplo da casa, corresponderia à escolha da massa corrida que será colocada nas paredes, as cores das tintas, os tipos de pisos. No caso do carro, se serão colocados bancos de tecido ou couro, qual será a cor do painel, vidros escuros ou claros e demais componentes.

Sendo assim, o framework trouxe a prática de evitar que tenhamos que fazer tarefas repetitivas, automatizando parte do trabalho. Pensando numa situação de desenvolvimento, se precisarmos criar um formulário de cadastro de usuário, ele sempre vai requerer algum tipo de validação como email e senha. O framework já terá essa validação pronta para ser utilizada.

2. Liste 4 Frameworks utilizados para programação Web selecionados por você, um para cada linguagem (Python, Java, PHP e Javascript). Descreva estes frameworks segundo seus propósitos e diferencial.

Python: FastAPI

O FastAPI é um framework de desenvolvimento web de alto desempenho para criação de APIs em Python. Ele é conhecido por sua velocidade, suporte a chamadas assíncronas e sintaxe declarativa. Com base no ASGI e no Python 3.7+, o FastAPI oferece uma forma fácil e produtiva de construir APIs rápidas e escaláveis. Ele suporta operações HTTP, possui integração com o pydantic para validação de dados e oferece recursos como documentação automática e suporte para autenticação e autorização. É uma escolha popular na comunidade Python para desenvolvedores que desejam criar APIs eficientes de maneira descomplicada.

Java: Spring

O Spring é um framework de desenvolvimento de aplicativos em Java que simplifica a criação de aplicativos empresariais escaláveis. Ele utiliza inversão de controle e injeção de dependência para gerenciar objetos e permite o desenvolvimento baseado em interfaces e anotações. Com diversos módulos e integração com outras tecnologias, o Spring oferece recursos avançados, como MVC, segurança e acesso a banco de dados. É amplamente utilizado devido à sua flexibilidade, modularidade e suporte ativo da comunidade.

PHP: Laravel

O Laravel é um framework de desenvolvimento web em PHP conhecido por sua elegância e eficiência. Ele segue o padrão MVC, possui um poderoso ORM chamado Eloquent e oferece recursos como autenticação, autorização e validação de formulários. Com uma comunidade ativa e suporte extenso, o Laravel simplifica o desenvolvimento de aplicativos web robustos e de alta qualidade.

JavaScript: ReactJS

O ReactJS é uma biblioteca JavaScript usada para criar interfaces de usuário interativas e dinâmicas. Com sua abordagem baseada em componentes reutilizáveis e o uso do Virtual DOM, o ReactJS simplifica o desenvolvimento de aplicativos web eficientes. Sua sintaxe JSX combina JavaScript com HTML, facilitando a escrita de código declarativo. Com uma comunidade ativa e suporte extenso, o ReactJS é uma escolha popular para construir interfaces modernas e responsivas.
